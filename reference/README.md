# Reference and source data

This folder holds **source and reference data** used to design or document the app. The running application does **not** read from here; it uses only `backend/data/pharmacy_circuit.json` as the baseline graph.

- **`databases/`** – APQC Process Classification Framework (Excel). Reference for process taxonomy.
- **`datasets/pharmacy/`** – Pharmacy circuit sources (TOHM, BPMN, Mermaid, citations). Used to derive the baseline graph; see `datasets/pharmacy/README.md` for details.
